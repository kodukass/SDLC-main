Git and GitHub

Git and Github are not the same thing. Git manages source code history and is a local file, that you have to install in your computer. Github is a hosting repository and web based. You use Git to code in local file and then push it to a web browser.

There are three main concepts in Git:

  • Repository: location, where the code history is stored.
  • Branch: “folder” in the repository containing commits.
  • Commit: different stages of code inside a branch.
